# TogoID Open — Chrome 拡張機能

> English version: [README.md](README.md)

ウェブページ上で生命科学データベースの ID を扱うための Chrome 拡張機能です。ホットキー一つで、選択した ID のデータベースレコードを開いたり、データベースの例示 ID を一覧・コピーしたりできます。

## 機能

- **Open selected** — ページ上で ID を選択してホットキーを押すと、対応するデータベースのレコードを新しいタブで開きます。正規表現パターンマッチングにより数百のデータベースに対応しています。URL の候補が複数あるデータベースでは、先頭の URL はワンクリックで開け、それ以外は「More」トグルから選択できます。
- **Show examples** — データベース名で検索し、例示 ID を一覧表示できます。各 ID をクリップボードにコピーしたり、レコードページを開いたりすることができます。最近使ったデータベースはリストの上部に表示されます。
- 両機能は、ひとつのポップアップウィンドウ内のタブとして提供されます。

## インストール

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」→ `togoid-open/` フォルダを選択

## 使い方

### ホットキー: `Alt+Shift+O`

- **テキスト選択中** → **Open selected** タブが開き、選択テキストの近くにポップアップが表示されます
- **テキスト未選択** → **Show examples** タブが開き、画面上部中央にポップアップが表示されます
- ホットキーをもう一度押す（または `Esc` キー、ポップアップ外クリック）で閉じます

### Open selected タブ

1. ページ上で ID を選択（例: `CHEMBL121649`、`GO:0005643`、`3PFQ`）
2. `Alt+Shift+O` を押す — またはタブを手動で切り替えて ID を入力・貼り付け
3. マッチするデータベースが候補行として表示される
   - URL が 1 つ → 行をクリックして開く
   - URL が複数 → 行クリックで先頭 URL を開く。**More ▾** で残りを展開

### Show examples タブ

1. データベース名を入力して絞り込む（または **Recent** / **All** から選ぶ）
2. データベース名をクリックして例示 ID リストを表示
3. 各 ID に **Copy** ボタンと **Open URL** ボタンが付いています
   - URL プレフィックスが複数あるデータベースはツールバーのドロップダウンで選択
   - ID の系列が複数ある場合（例: PDB の大文字・小文字）はアコーディオン表示
4. **← Back** でデータベース一覧に戻る

### ホットキーの変更

`chrome://extensions/shortcuts` を開き、「TogoID Open」の欄で変更してください。

## ID → URL の変換

各データベースの正規表現中の名前付きキャプチャグループが、URI プレフィックスに付加する部分を決定します：

| 正規表現パターン | 入力例 | 解決される ID |
|----------------|--------|-------------|
| `(?<id>\d{7})` | `GO:0005643` | `0005643` |
| `(?<id1>…)\|(?<id2>…)` | `P12345` | 最初にマッチしたグループ |
| `([0-9][A-Za-z0-9]{3})` | `3PFQ` | `3PFQ`（グループ 1） |

## データベース定義ファイル

| ファイル | 由来 | 備考 |
|---------|------|------|
| `dataset.yaml` | TogoID プロジェクト | 外部由来。手動編集不要 |
| `dataset-local.yaml` | TogoID Open 独自 | TogoID、EBI OLS など独自追加分 |

ユーザーによる追加・無効化設定は `chrome.storage.sync` に別途保存されるため、YAML ファイルは変更されません。

`dataset-local.yaml` へのデータベース追加例：

```yaml
my_db:
  label: My Database
  regex: '^(?<id>PREFIX\d+)$'
  prefix:
    - label: main
      uri: 'https://example.com/record/'
  examples:
    - ["PREFIX001", "PREFIX002", "PREFIX003"]
```

## オプション設定画面

拡張機能アイコンを右クリック → 「オプション」から開きます。

| 機能 | 説明 |
|------|------|
| データベース一覧 | 折り畳み可能、名前・キーで絞り込み検索 |
| 有効/無効の切り替え | DB 単位・prefix 単位でチェックボックス操作 |
| prefix の追加 | 各デフォルト DB カードの「＋ prefix」ボタン |
| DB の追加 | 画面下部のフォームから |
| 編集 / 削除 | カスタム DB カードに表示 |
| 言語切り替え | サイドバーの EN / JA ボタン |
| ホットキー | `chrome://extensions/shortcuts` へのリンク |

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `manifest.json` | 拡張機能の設定（権限・ホットキー・オプションページ） |
| `dataset.yaml` | デフォルトのデータベース定義（TogoID プロジェクト由来） |
| `dataset-local.yaml` | TogoID Open 独自の追加データベース定義 |
| `databases.js` | YAML 読み込み・名前付きキャプチャ対応の URL 生成・候補絞り込み |
| `background.js` | Service Worker：ホットキー受信・content script 注入 |
| `content.js` | 統合タブポップアップ（Open selected + Show examples） |
| `options.html/css/js` | オプション設定画面 |
| `i18n.js` | オプション画面の EN/JA 文字列テーブル |
| `CHANGELOG.md` | バージョン履歴 |
