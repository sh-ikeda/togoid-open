# TogoID Open — Chrome 拡張機能

> English version: [README.md](README.md)

ウェブページ上でデータベース ID を選択し、対応するレコードページを直接開く Chrome 拡張機能です。

## インストール

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」→ `togoid-open/` フォルダを選択

## 使い方

### ホットキー: `Alt+Shift+O`

**テキストを選択している場合：**
1. ID テキストを選択（例: `CHEMBL121649`、`GO:0005643`、`3PFQ`）
2. `Alt+Shift+O` を押す
3. 選択テキストの近くにポップアップが表示される
   - 候補が 1 件 → クリックで即オープン
   - prefix が複数ある DB → `▶ DB名 (N)` をクリックして展開 → prefix を選択

**テキストを選択していない場合：**
1. `Alt+Shift+O` を押す
2. データベースブラウザのポップアップが表示される
3. 文字を入力してデータベースを絞り込み → クリックして選択
4. 例示 ID のリストが表示される（**コピー** / **URL を開く** ボタン付き）
   - ID の系列が複数ある場合はアコーディオン表示
   - prefix が複数ある DB の「URL を開く」では選択肢が提示される

`Esc` またはポップアップ外クリックで閉じます。

### ホットキーの変更

`chrome://extensions/shortcuts` からいつでも変更できます。

## ID → URL の変換

正規表現の名前付きキャプチャグループが URI プレフィックスに付加する部分を決定します：

| パターン | 入力例 | 解決される ID |
|---------|--------|-------------|
| `(?<id>\d{7})` | `GO:0005643` | `0005643` |
| `(?<id1>…)\|(?<id2>…)` | `P12345` | 最初にマッチしたグループ |
| `([0-9][A-Za-z0-9]{3})` | `3PFQ` | `3PFQ`（グループ 1） |

## データベースの追加

デフォルト定義は `dataset.yaml` に記述します。ユーザーによるカスタマイズは `chrome.storage.sync` に別途保存されるため、デフォルトファイルは変更されません。

```yaml
chebi:
  label: ChEBI
  regex: '^CHEBI:(?<id>\d+)$'
  prefix:
    - label: EBI
      uri: 'https://www.ebi.ac.uk/chebi/searchId.do?chebiId=CHEBI:'
  examples:
    - ["CHEBI:15422","CHEBI:17234","CHEBI:16541"]
```

## オプション設定画面

拡張機能アイコンを右クリック → 「オプション」から開きます。

| 機能 | 説明 |
|------|------|
| データベース一覧 | 折り畳み可能、名前・キーで絞り込み検索 |
| 有効/無効の切り替え | DB 単位・prefix 単位でチェックボックス操作 |
| prefix の追加 | 各デフォルト DB カードの「＋ prefix」ボタン |
| DB の追加 | 画面下部のフォームから |
| 編集 / 削除 | カスタム DB カードに表示 |
| 言語切り替え | サイドバーの EN / JA ボタン |
| ホットキー | `chrome://extensions/shortcuts` へのリンク |

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `manifest.json` | 拡張機能の設定（権限・ホットキー・オプションページ） |
| `dataset.yaml` | デフォルトのデータベース定義 |
| `databases.js` | YAML 読み込み・名前付きキャプチャ対応の URL 生成・候補絞り込み |
| `background.js` | Service Worker：ホットキー受信・content script 注入 |
| `content.js` | 開くポップアップとブラウザポップアップの UI |
| `options.html/css/js` | オプション設定画面 |
| `i18n.js` | オプション画面の EN/JA 文字列テーブル |
